ShopNexa is a modern e-commerce frontend built with React, Redux, and Tailwind CSS. 
It currently includes user authentication (login/signup) and a responsive product listing page with filters. 
Future updates will integrate a full MERN stack backend (Node.js, Express, MongoDB) for dynamic product management and order processing.
